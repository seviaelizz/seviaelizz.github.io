Materi Simulasi Orbit Planet

1. Dasar Hukum Gravitasi Newton

Hukum gravitasi Newton menyatakan bahwa gaya tarik-menarik antara dua benda sebanding dengan hasil kali massa kedua benda dan berbanding terbalik dengan kuadrat jarak antara keduanya:

Persamaan:
F = G * (m1 * m2) / r^2

  • F: Gaya gravitasi antara dua benda
  • G: Konstanta gravitasi universal (6.674 × 10-11 Nm²/kg²)
  • m1, m2: Massa kedua benda
  • r: Jarak antara kedua benda

2. Konsep Simulasi Orbit

Simulasi ini memodelkan pergerakan planet mengitari benda pusat (contohnya Matahari) menggunakan percepatan gravitasi.

3. Metode Integrasi Euler

Metode Euler digunakan untuk memperbarui posisi dan kecepatan planet berdasarkan persamaan diferensial.

  • Posisi: x(t+dt) = x(t) + v(t) * dt
  • Kecepatan: v(t+dt) = v(t) + a(t) * dt

Dengan percepatan a yang diperoleh dari hukum gravitasi:

a = -G * M / r^3 * r

4. Langkah Simulasi

  1. Inisialisasi posisi awal, kecepatan awal, dan parameter lainnya.
  2. Hitung percepatan gravitasi pada setiap langkah waktu.
  3. Perbarui kecepatan dan posisi menggunakan metode Euler.

5. Contoh Kode Pseudocode


# Parameter awal
x = [x0, y0]
v = [vx0, vy0]
M = 1.989e30
G = 6.674e-11
dt = 1e5
num_steps = 10000

# Iterasi metode Euler
for i in range(num_steps):
    r = sqrt(x[0]*2 + x[1]*2)
    a = -G * M / r**3
    ax = a * x[0]
    ay = a * x[1]
    
    # Perbarui kecepatan
    v[0] += ax * dt
    v[1] += ay * dt
    
    # Perbarui posisi
    x[0] += v[0] * dt
    x[1] += v[1] * dt
    
    print(f"Step {i}: Posisi = {x}")

6. Visualisasi Orbit

Dengan menggunakan pustaka seperti Matplotlib di Python, orbit planet dapat divisualisasikan sebagai lintasan elips jika parameter awal ditentukan dengan benar.

7. Kelebihan dan Kekurangan Metode Euler

  • Kelebihan: Sederhana dan mudah diimplementasikan.
  • Kekurangan: Kurang akurat untuk langkah waktu yang besar dan bisa menjadi tidak stabil.

Solusi: Gunakan metode seperti Runge-Kutta untuk hasil lebih stabil.
